# ⚡ Power Supply E-Market

An interactive **frontend-based e-commerce platform** designed for buying electronic products with a smooth user experience.
This project simulates a real-world online marketplace with product management, cart functionality, and payment flow.

---

## 🚀 Features

* 🛍️ **Product Listing**

  * Displays available electronics dynamically
  * Clean card-based UI design

* ➕ **Add to Cart**

  * Users can add products to their cart
  * Cart data is stored using localStorage

* 🧾 **Cart Management**

  * View selected items
  * Remove items from cart
  * Calculate total cost

* 💳 **Payment Simulation**

  * M-Pesa integration (simulated)
  * PayPal option
  * Bank transfer option

* 📦 **Order Handling**

  * Track orders
  * Mark completed orders

* ⚙️ **Admin Product Control**

  * Add products dynamically
  * Delete products
  * Products persist using localStorage

* 📱 **Responsive Sidebar Navigation**

  * Toggle menu for better mobile experience

---

## 🛠️ Technologies Used

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **LocalStorage API**

---

## 🧠 How It Works

This project uses **localStorage** as a temporary database:

* Products are stored in:

  ```
  goods-view
  ```

* Cart items are stored in:

  ```
  cart-view
  ```

* Orders and payments are handled dynamically in the frontend

## ⚠️ Limitations

* ❌ No backend (data is not secure)
* ❌ Data can be modified from browser console
* ❌ Not suitable for real payments
